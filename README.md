
# Passgen

Passgen is a simple password  generator using golang.


## Build

Build CLi with golang

```bash
 $ git clone https://github.com/Nouments/passgen/
 $ cd passgen
 $ go build -o passgen .
```
    
## Demo

Usage of binary (.exe for windows)
```bash
$ chmod +x passgen
$ ./passgen
```
## Result


```bash
$ chmod +x passgen
$ ./passgen
Entered password lenth: 
12
Generated password: Z2{zLBx+^hSe

```
